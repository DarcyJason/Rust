# 基本数据类型

## 1.整数类型
|长度|有符号整数|无符号整数|
|---|---------|--------|
|8  |i8       |u8      |
|16 |i16      |u16     |
|32 |i32      |u32     |
|64 |i64      |u64     |
|128|i128     |u128    |
|arch|isize    |usize   |

## 2.浮点数类型
|长度|有符号浮点数|
|---|----------|
|32 |f32       |
|64 |f64       |
|128|f128      |

## 3.布尔类型
|布尔类型|真   |假   |
|-------|----|-----|
|bool   |true|false|

## 4.字符类型
```rust
let a: char = 'a';
println!("{}", a);
```

## 5.字符串切片类型
```rust
let a: &str = "hello";
println!("{}", a);
```

## 6.字符串类型
```rust
let a: String = String::from("hello");
println!("{}", a);
```
